The `Radio` component allows users to select a single option from a set of options. It is typically used within a `RadioGroup` to provide a group of related options.

## Radio Props

| Name          | Type                                      | Required | Description                                                                                                                 |
| ------------- | ----------------------------------------- | -------- | --------------------------------------------------------------------------------------------------------------------------- |
| label         | string                                    | No       | The label prop provides a label or title for the Radio component, describing the input field's purpose or expected content. |
| items         | `Array<{ label: string, value: string }>` | Yes      | An array of objects representing each radio option.                                                                         |
| direction     | `Enum { HORIZONTAL, VERTICAL }`           | Yes      | Specifies the direction of the radio options display.                                                                       |
| selectedValue | string                                    | No       | The value of the currently selected radio option.                                                                           |
| onValueChange | function                                  | Yes      | A callback function invoked when the selected value changes.                                                                |
| classNames    | string                                    | No       | Additional CSS classes for custom styling.                                                                                  |

#### **Notes :**

- Ensure each Radio component has a unique `value` prop value.
- Utilize the `value` prop to associate a specific value with each radio option.
- The `label` prop provides the text or number displayed next to the radio button.
- Use the optional `classNames` prop for custom styling using CSS classes.

### Code

```tsx
import { Radio, Button, ButtonVariants } from "@bmesol/bmes-components";

export const SampleRadio = () => {
  const [gender, setGender] = useState("Male");
  const genderRadioList = [
    { label: "Male", value: "Male" },
    { label: "Female", value: "Female" },
  ];
  return (
    <>
      <div className="d-flex mt-4">
        <div className="text-success fw-bold">Select Gender : </div>
        <Radio
          label="Select Gender"
          items={genderRadioList}
          direction={Directions.HORIZONTAL}
          selectedValue="Male"
          onValueChange={setGender}
        />
      </div>

      <Button
        label="Show Gender"
        variant={ButtonVariants.OUTLINE_SECONDARY}
        onClick={() => {
          alert("You selected " + gender + " as your Gender");
        }}
        classNames="mt-3 mb-4"
      />
    </>
  );
};
```
