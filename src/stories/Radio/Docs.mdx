The `Radio` component allows users to select a single option from a set of options. It is typically used within a `RadioGroup` to provide a group of related options.

## RadioGroup Props

| Prop          | Type      | Description                                                                                                                       | Required |
| ------------- | --------- | --------------------------------------------------------------------------------------------------------------------------------- | -------- |
| defaultValue  | string    | The default value for the RadioGroup. If provided, it sets the initially selected radio button.                                   | No       |
| onValueChange | function  | A callback function invoked when the selected value of the RadioGroup changes. It receives the new selected value as a parameter. | Yes      |
| children      | ReactNode | The children of the RadioGroup component. These are typically the individual radio button components.                             | Yes      |
| classNames?   | string    | Additional CSS classes for styling purposes.                                                                                      | No       |

## Radio Props

| Prop        | Type   | Description                                         | Required |
| ----------- | ------ | --------------------------------------------------- | -------- |
| id          | string | Unique identifier for the radio input.              | Yes      |
| value       | string | Value associated with the radio option.             | Yes      |
| label       | string | Text or number to display next to the radio button. | Yes      |
| classNames? | string | Additional CSS classes for styling purposes.        | No       |

#### **Notes :**

- Ensure each Radio component has a unique `id` prop value.
- Utilize the `value` prop to associate a specific value with each radio option.
- The `label` prop provides the text or number displayed next to the radio button.
- Use the optional `classNames` prop for custom styling using CSS classes.

### Code

```tsx
import { RadioGroup, Radio, Button, ButtonVariants } from "@bmesol/bmes-components";

export const SampleRadio = () => {
  const [gender, setGender] = useState("Male");
  return (
    <>
      <RadioGroup
        defaultValue="Male"
        onValueChange={setGender}
        classNames="d-flex mt-4"
      >
        <div className="mb-2 text-primary">Select Gender : </div>
        <Radio id="1" value="Male" label={"Male"} />
        <Radio id="2" value="Female" label={"Female"} />
      </RadioGroup>

      <Button
        label="Show Gender"
        variant={ButtonVariants.OUTLINE_SECONDARY}
        onClick={() => {
          alert("You selected " + gender + " as your Gender");
        }}
        classNames="mt-3 mb-4"
      />
    </>
  );
};
```
