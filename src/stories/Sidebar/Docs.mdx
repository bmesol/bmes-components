The Sidebar component is designed to create a collapsible menu interface for navigation or displaying additional information. It offers flexibility and customization options for building intuitive user interfaces.

## Props

* **Sidebar**: The content of the sidebar.
        * **`id`** : This id serves as a unique identifier for the Sidebar.
* **SidebarTrigger**: The trigger element to open the sidebar.
        * **`sidebarId`** : This sidebarId specifies the id of the sidebar that the trigger is associated with. It helps in identifying which sidebar should be opened or closed when the trigger is activated.
* **SidebarHeader**: Content displayed at the top of the sidebar.
* **SidebarMenuItems**: The list of menu items displayed in the sidebar.
* **SidebarMenuItem**: Represents an item in the sidebar menu.
        * **`href`**: The URL to navigate to when the menu item is clicked. 
        * **`icon`**: An optional icon to display alongside the menu item. 
        * **`target`**: Optional target attribute for the link.
* **SidebarFooter**: Content displayed at the bottom of the sidebar.

### Code

```tsx

import { Sidebar, SidebarTrigger, SidebarHeader, SidebarMenuItems, SidebarMenuItem, SidebarFooter, Button, ButtonVariants } from "@bmesol/bmes-components";

export const SampleSidebar = () => {
  return (
    <>
        <SidebarTrigger sidebarId="menu">
          <Button variant={ButtonVariants.PRIMARY} label="Open" />
        </SidebarTrigger>

        <Sidebar id="menu">
          <SidebarHeader>
            <div className="d-flex flex-column justify-content-center align-items-center text-secondary my-3">
              <ProfileIcon />
              <div className="mt-1">Tony Stark</div>
            </div>
          </SidebarHeader>
          <SidebarMenuItems>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<UserIcon />}>Users</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<ContactsIcon />}>Contacts</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<ClientsIcon />}>Clients</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<SitesIcon />}>Sites</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<WorkersIcon />}>Workers</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<SuppliersIcon />}>Suppliers</SidebarMenuItem>
          </SidebarMenuItems>
          <SidebarFooter>
            <div>&copy; BM e-Solutions</div>
          </SidebarFooter>
        </Sidebar>
    </>
  );
};
```
