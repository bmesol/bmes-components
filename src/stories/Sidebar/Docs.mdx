The Sidebar component is designed to create a collapsible menu interface for navigation or displaying additional information. It offers flexibility and customization options for building intuitive user interfaces.

## Props

1. **Sidebar**: The content of the sidebar.
        1. **`id`** : This id serves as a unique identifier for the Sidebar.
2. **SidebarTrigger**: The trigger element to open the sidebar.
        1. **`sidebarId`** : This sidebarId specifies the id of the sidebar that the trigger is associated with. It helps in identifying which sidebar should be opened or closed when the trigger is activated.
3. **SidebarHeader**: Content displayed at the top of the sidebar.
4. **SidebarMenuItems**: The list of menu items displayed in the sidebar.
5. **SidebarMenuItem**: Represents an item in the sidebar menu.
        1. **`href`**: The URL to navigate to when the menu item is clicked. 
        2. **`icon`**: An optional icon to display alongside the menu item. 
        3. **`target`**: Optional target attribute for the link.
6. **SidebarFooter**: Content displayed at the bottom of the sidebar.

## Import

```tsx
import { Sidebar, SidebarTrigger, SidebarHeader, SidebarMenuItems, SidebarMenuItem, SidebarFooter } from "@bmesol/bmes-components"
```

## Usage

```tsx
export const SampleMenu = () => {
  return (
    <>
        <SidebarTrigger sidebarId="navMenu">
            <button>Open Sidebar<button/>
        </SidebarTrigger>

        <Sidebar id="navMenu">
            <SidebarHeader>
                <div>Sidebar Header</div>
            </SidebarHeader>
            <SidebarMenuItems>
                <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<UserIcon />} target="blank">
                    Item-1 with target
                </SidebarMenuItem>
                <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<ContactsIcon />}>
                    Item-2 without target
                </SidebarMenuItem>
            </SidebarMenuItems>
            <SidebarFooter>
                <div>Sidebar Footer</div>
            </SidebarFooter>
        </Sidebar>
    </>
  )
}
```

import { Sidebar, SidebarTrigger, SidebarHeader, SidebarMenuItems, SidebarMenuItem, SidebarFooter } from "@bmesol/bmes-components"
import { Button, ButtonVariants } from "@bmesol/bmes-components"

export const SampleMenu = () => {
  return (

            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<UserIcon />}>Users</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<ContactsIcon />}>Contacts</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<ClientsIcon />}>Clients</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<SitesIcon />}>Sites</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<WorkersIcon />}>Workers</SidebarMenuItem>
            <SidebarMenuItem href={"https://www.bmesolutions.in/"} icon={<SuppliersIcon />}>Suppliers</SidebarMenuItem>
          </SidebarMenuItems>

          </SidebarFooter>
        </Sidebar>
    </>
  );
};
