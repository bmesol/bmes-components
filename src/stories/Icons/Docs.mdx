# Icon

The `Icon` component is a customizable UI element that displays icons of various sizes. It provides a selection of predefined icon sizes and allows customization of the icon color and click behavior.

## Props

| Props    | Type      | Required | Description                                                                                                     |
| -------- | --------- | -------- | --------------------------------------------------------------------------------------------------------------- |
| size     | IconSizes | No       | The size of the icon. Options: `IconSizes.SM`, `IconSizes.MD`, `IconSizes.LG`, `IconSizes.XL`, `IconSizes.XXL`. |
| onClick  | function  | No       | Callback function invoked when the icon is clicked.                                                             |
| color    | string    | No       | The color of the icon.                                                                                          |
| isActive | boolean   | No       | Specifies if the icon is active or not.                                                                         |

## Usage

Open the Icons file and click on the Icon. Past it on your code.

## Sample Code

```jsx
import { ClientsIcon, IconSizes } from "@bmesol/bmes-components";

const MyComponent = () => {
  return (
    <>
      Client <ClientsIcon size={IconSizes.SM} color="red" onClick={() => alert("Clicked")}/>
    </>
  );
};
```
