The `Textarea` component is a customizable text input field that allows users to enter multi-line text. It provides various props for configuring its behavior and appearance.

## Props

| Props       | Type     | Required | Description                                                                                                                                                         |
| ----------- | -------- | :------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| label       | string   |    No    | A label for the textarea, describing the purpose of the input.                                                                                                      |
| length      | number   |   Yes    | The maximum length of the text that can be entered into the textarea.                                                                                               |
| value       | string   |   Yes    | The current value of the textarea.                                                                                                                                  |
| onChange    | Function |   Yes    | A callback function that is triggered when the textarea value changes. It takes one argument, currentValue, which is the current value of the textarea as a string. |
| placeholder | string   |    No    | A placeholder text displayed in the textarea when it is empty.                                                                                                      |
| isDisabled  | boolean  |    No    | Specifies whether the textarea is disabled or not.                                                                                                                  |
| classNames  | string   |    No    | Additional CSS classes to be applied to the textarea.                                                                                                               |

### Notes

- The `length` prop is used to enforce a maximum character limit for the input. Ensure it is set appropriately based on your requirements.

- The `onChange` callback is essential for capturing user input and updating the textarea's value accordingly.

- When `isDisabled` is set to `true`, the textarea becomes non-editable, and its appearance may change to indicate its disabled state.

- You can apply custom styles to the textarea using the `classNames` prop.

### Code

```tsx
import { Textarea, Button, ButtonVariants } from "@bmesol/bmes-components";

export const SimpleTextarea = () => {
  const [value, setValue] = useState("");
  return (
    <>
      <Textarea
        label="Notes"
        length={1000}
        value={value}
        onChange={(inputValue: string) => setValue(inputValue)}
        placeholder="Enter value here"
      />
      <Button
        variant={ButtonVariants.PRIMARY}
        label="Submit"
        onClick={() => alert("Your Notes : " + value)}
      />
    </>
  );
};
```
