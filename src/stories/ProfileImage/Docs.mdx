The `Profile Image` component displays a user's profile image with customizable size options. If the `imgURL` is not provided, the `alt` first two sentences first letters will be displayed as the profile image.

## Props

| Props  | Type              | Required | Description                                                                                                                             |
| ------ | ----------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| imgURL | string            | No       | The URL of the image to be displayed.                                                                                                   |
| alt    | string            | No       | The alternative text for the image (for accessibility).                                                                                 |
| size   | ProfileImageSizes | No       | The size of the profile image. Options: `ProfileImageSizes.SM`, `ProfileImageSizes.MD`, `ProfileImageSizes.LG`, `ProfileImageSizes.XL`. |

## Usage

```tsx
import { ProfileImage, ProfileImageSizes } from "@bmesol/bmes-components";

export const SampleProfile = () => {
  return (
    <>
      <ProfileImage
        imgURL="path_to_image.jpg"
        alt="Tony stark"
        size={ProfileImageSizes.SM}
      />
      Tony stark
    </>
  );
};
```
