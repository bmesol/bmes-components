The Badge component is designed to display a badge with a label.

## Props

| Props      | Type   | Required | Description                                                                                           |
| ---------- | ------ | -------- | ----------------------------------------------------------------------------------------------------- |
| label      | string | Yes      | The text to be displayed inside the badge.                                                            |
| classNames | string | No       | Offers the flexibility to apply additional custom CSS classes to the badge for styling customization. |
