The `SearchBox` component provides a user interface element for inputting search queries and triggering search actions within your application. It can be easily integrated into your project to enhance search functionality.

## Props

| Name     | Type                            | Required | Description                                                                                                                                                                 |
| -------- | ------------------------------- | :------: | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| onSearch | `(searchValue: string) => void` |   Yes    | A function that will be called when the user initiates a search. It takes one parameter: `searchValue`, a string representing the current search query entered by the user. |

## Usage

```tsx
import { SearchBox } from "@bmesol/bmes-components";

export const SampleSearchBox = () => {
  const handleSearchButton = (searchValue: string) => {
    alert("You Searched : " + searchValue);
  };
  return (
    <>
      <SearchBox onSearch={handleSearchButton} />
    </>
  );
};
```
